

https://nodejs.org/en/

Install Node.js for windows run https://nodejs.org/dist/v14.17.3/node-v14.17.3-x64.msi


Creaza folderul http-server pe hdd c:

unzip https://github.com/lrbastro/deceneu/blob/master/codNB.zip?raw=true in folderul http-server

Deschizi command prompt si rulezi npm install --global http-server si npm install http-server

c:\users\mytv> npm install --global http-server

c:\users\mytv> npm install http-server

si apoi rulezi cd / si cd http-server

c:\users\mytv> cd /

c:\> cd http-server

c:\http-server>cd NBmsx

c:\http-server\NBmsx> http-server --cors --log-ip
Starting up http-server, serving ./
Available on:
  http://10.0.0.01:8080
  http://127.0.0.1:8080
